# tmux_conf
.tmux.conf file

## INSTALL  
Use curl:

    curl -o - https://raw.githubusercontent.com/motokii/tmux_conf/master/install.sh | sh

  
Use wget:  

    wget -O - https://raw.githubusercontent.com/motokii/tmux_conf/master/install.sh | sh

## prefix
prefix: `Control-B` -> `Control-Q`
